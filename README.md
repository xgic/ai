# XGIC AI

Public hub for XGIC open-source multi-repository standards, architecture decision records (ADRs), and high-level project orientation.

This repository helps humans and AI agents (including Grok Build) understand how XGIC's public GitHub repositories fit together: libraries, frameworks, GHCR images, dev-container producers, and templates.

## What lives here
- Sanitized ADRs under docs/adr/
- High-level Python namespace and module registry under docs/
- Orientation for public XGIC OSS strategy (no private coordination details)
